\abstract
    {Основная цель проекта - понимание механизмов регуляции движения конечностей нейронами головного мозга. Для построения зависимостей необходимо подробное признаковое описание сигналов головного мозга и связанных с ними поведенческих реакций. Данная работа сконцентрирована на построении модели (объектов выборки, признаков) на основе временного ряда, полученного по результатам исследований активности нейронов (основной источник - электрокортикограмма). С помощью модели необходимо проверить гипотезу, состоящую в том, что перемещение зон активности головного мозга является информативным признаком при решении задачи BCI. В более ранних исследованиях для прогнозирования использовались частоты, присутствующие в сигнале электрокортикограммы. Предлагается заменить частоты другим набором признаков, используя пространственную структуру сигналов, снятую датчиками. В задаче используются данные ECoG, собранные на основе исследований корреляции активности нейронов головного мозга обезъян и их поведения при различных условиях. Данные представлены на странице Neurotycho.org.

    {Spatial Signal Decoding for Brain Computer Interface.}
\authorEng
    {Shiyanov~V.\,A., Bolobolova~N.\,A., Samokhina~A.\,M., Mokrypolo~M.\,N.}
%\organizationEng

\abstractEng
    {The project is mainly aimed at understanding neural regulation of limb movement. To build dependencies, one needs a significant description of brain signals and related behavioral responses. This work is focused on building a model (objects, features), based on time series obtained from the results of the neural activity research (the main source is ECoG). The model is expected to help in testing the hypothesis that the movement of brain activity areas is an informative feature for solving the BCI problem. In earlier studies, the frequencies present in the signal of the electrocorticogram were used to predict. Here frequencies are suggested to be replaced with another set of features, using spatial structure of signals, taken by sensors. Data used here have been collected using monkeys' brain neural activity and their behavior dependencies studies under various conditions. Dataset is submitted on Neurotycho.org.
